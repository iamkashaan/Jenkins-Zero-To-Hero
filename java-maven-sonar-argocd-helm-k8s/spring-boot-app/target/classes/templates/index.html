Get Started in Minutes

Ready to start building your Spring Boot application?

Using Spring Initializr

Generate a new project with just what you need: 

$ curl https://start.spring.io/starter.tgz \
  -d dependencies=web,data-jpa \
  -d packageName=com.example \
  -d name=myapp \
  -d javaVersion=17 | tar -xzvf -

Or visit start.spring.io for a web interface.

Manual Setup

1. Add Spring Boot dependencies to your build.gradle or pom.xml 

<!-- Maven -->
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
</dependency>

2. Create your main application class with @SpringBootApplication

3. Run with: ./mvnw spring-boot:run or ./gradlew bootRun
